Installation

Downloading WiiMart varies on what platform you are on.

Real Console:

Download the WiiMart WAD and the patched IOS WAD for your console type (Original Wii or Wii U's Wii Mode).
We recommend YAWMME for installing WADs.
If you receive an error such as

-1022
or
-1029
, you can delete the original Wii Shop Channel using Any Title Deleter MOD and then try installing the WADS again.

Dolphin (with a Real Wii NAND backup):

Install both the WiiMart WAD and the patched IOS WAD (for Wii) by going to
Tools
and selecting
Install WAD.

You can then go to the Wii Menu by going to
Tools
and then
Load Wii System Menu 4.3X.
